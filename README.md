# Retro Balls

A real-time browser-based shooter where you battle three enemy balls in an arena.

## How to Play

| Key | Action |
|-----|--------|
| `←` `→` | Move left / right |
| `↑` | Jump |
| `Space` | Shoot |

## Gameplay

- You control the green ball. Three enemy balls (red, yellow, purple) fight back in real time.
- Shoot enemies to deal damage. Each ball has a health bar — deplete it to eliminate them.
- Enemies move toward you, jump, and shoot autonomously.
- The game ends when you destroy all enemies (**Victory**) or your health reaches zero (**Defeat**).

## Scoring

| Event | Points |
|-------|--------|
| Hit an enemy | +100 |
| Kill an enemy | +1000 |
| Take a hit | -10 |

## Running the Game

Open `index.html` in any modern browser — no build step or server required.
